# -Provide-Insights-for-a-Strategic-Merger-in-the-OTT-Domain
Developed an interactive Power BI dashboard analyzing the merger of OTT platforms Jotstar and Liocinema
for Codebasics Resume Project Challenge 14. Performed data transformation using Power Query to clean,
merge, and standardize datasets from both platforms. Built a robust data model with relationships across user,
content, and subscription tables. Leveraged DAX to create dynamic measures for watch time, revenue growth,
and churn rate. Implemented context-aware slicers for genre, language, region, and plan type to enable
interactive filtering. Visualizations revealed top-performing content, subscription trends, and retention
drivers, empowering stakeholders with actionable insights to optimize pricing, content strategy, and user
engagement post-merger.
